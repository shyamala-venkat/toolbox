FAQ

Is it really free?

Yes. 20 tools are free forever with no limits, no accounts, no ads. Pro unlocks all 44 tools for a one-time $30 payment.

Do my files get uploaded anywhere?

No. Every tool runs 100% on your machine. The app makes zero network calls. Your files, your data, your privacy.

Does it work offline?

Yes. ToolBox works without internet. It's a native desktop app — not a website wrapper.

Where are my API keys stored?

In your operating system's secure keychain (macOS Keychain / Windows Credential Manager). Never in files, never in localStorage.

What's the tech stack?

Tauri 2 (Rust backend) + React 19 + TypeScript. Image processing uses mozjpeg and oxipng for best-in-class compression. Not Electron — 10x smaller and faster.
